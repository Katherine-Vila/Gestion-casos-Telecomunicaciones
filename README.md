## 🎯 Objetivo
Desarrollar una aplicación (módulo Escritorio y Web) que permita:  
- Apertura de casos por jefes de área funcional.  
- Asignación de programadores y probadores.  
- Seguimiento mediante bitácoras y porcentajes de avance.  
- Validación de cambios por probadores (aprobación o rechazo).  
- Generación de reportes de casos cumplidos, en desarrollo y rechazados.  
- Vista general del avance del proyecto (solo en módulo Web).  

---

## 🔑 Roles de Usuario
- **Administrador:** gestiona departamentos y jefes.  
- **Jefes de área funcional:** solicitan casos y monitorean progreso.  
- **Jefes de desarrollo:** aceptan/rechazan solicitudes y asignan programadores.  
- **Programadores:** actualizan bitácoras y porcentajes de avance.  
- **Probadores:** aprueban o rechazan casos.  

---

## 🛠️ División del Trabajo
- **Persona 1 (Modelo y BD):** Clases en Java + script SQL.  
- **Persona 2 (Flujo y Lógica):** Métodos principales + reglas de negocio.  
- **Persona 3 (Interfaz y Reportes):** Formularios básicos + reportes.  

---

## 🚀 Cómo ejecutar
1. Importar las clases en un proyecto Java.  
2. Configurar la base de datos MySQL con el script proporcionado.  
3. Ejecutar el archivo `Main.java` para probar el flujo básico de casos.  
4. Usar los formularios simples para simular la interfaz y generar reportes.
